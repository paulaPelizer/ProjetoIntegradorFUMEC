Frontend Emprende+

Projeto React + Vite + Tailwind para interface com alternância de tema claro/escuro.

## Rodar localmente

```bash
npm install
npm run dev
```

Abra o endereço exibido no terminal, normalmente `http://localhost:5173/`.

## Observações

- `node_modules/`, `dist/` e arquivos `.env` não devem ser versionados.
- O projeto foi limpo para remover referências de origem/exportação do Figma.
